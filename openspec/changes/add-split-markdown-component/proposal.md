# Proposal: Add Split-Pane Markdown Component

## Summary

Implement a simple dual-pane Markdown component containing `MDEditor` on the left and `MDPreview` on the right. A draggable separator will be placed between the two columns to allow users to adjust the width ratio dynamically.

## Motivation

To provide a seamless markdown editing experience where users can see a live preview of their content. The draggable separator allows flexibility for viewing longer lines of code or focusing on the visual output, which is particularly useful on different screen sizes.
